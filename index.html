💡 スムーズな登録方法（AIを使う場合）
  1. 選挙管理委員会などから受け取った候補者名簿（紙・PDF・画像）を用意する
  2. Claude や ChatGPT などのAIに、その資料の画像や内容を貼り付ける
  3. 下記の「AIへの依頼文」をそのままコピーしてAIに渡す
  4. AIが出力したCSVテキストをメモ帳などに貼り付け、.csv形式で保存する
  5. 下の「ファイルを選択」からアップロードして登録する
🤖 AIへの依頼文（コピーして使ってください）
添付の候補者名簿データをもとに、以下のCSVフォーマットに変換してください。 【フォーマット】 候補者ID,パスワード,氏名,選挙区分,選挙区名,比例名簿順位 【ルール】 - 候補者IDは候補者の苗字のローマ字読み+3桁の連番にする（例：安野 → anno001、山田 → yamada001） - パスワードは「pass」+候補者IDの番号（例：pass001）にする - 選挙区分は以下の3種類のいずれかを英語で記入する 　比例代表 → proportional 　小選挙区 → single 　小選挙区（重複立候補）→ single_double - 選挙区名は「東京7区」「九州ブロック（比例）」のように記入する（比例の場合はブロック名） - 比例名簿順位は「proportional」または「single_double」の場合のみ数字を入力する。「single」の場合は空欄にする - ヘッダー行は不要、データ行のみ出力する - 文字コードはUTF-8 【出力例】 anno001,pass001,安野たかひろ,single,東京7区, yamada001,pass002,山田太郎,proportional,東京ブロック（比例）,1 suzuki001,pass003,鈴木花子,single_double,東京12区,3
📋 CSVフォーマット（直接作成する場合）
列の順番: ID, パスワード, 氏名, 選挙区分, 選挙区名, 比例名簿順位
選挙区分: proportional（比例）／single（小選挙区）／single_double（重複）　※比例名簿順位は比例・重複のみ記入
